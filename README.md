# Task Tracker

Sistema simples em Python para gerenciar tarefas pelo terminal.

## Problema
Muitas pessoas precisam organizar tarefas simples do dia a dia, como estudos, compromissos e atividades pessoais.

## Solução
Este projeto permite adicionar, listar, concluir e remover tarefas usando comandos no terminal.

## Funcionalidades
- Adicionar tarefa
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa
- Salvar tarefas em arquivo JSON

## Tecnologias utilizadas
- Python
- JSON
- Git e GitHub

## Como usar
```bash
python main.py add "Estudar Python"
python main.py list
python main.py done 1
python main.py remove 1
```

## Aprendizados técnicos

- Leitura e escrita de arquivos JSON
- Uso de argumentos pelo terminal (sys.argv)
