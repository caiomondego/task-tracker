# Task Tracker

Um projeto simples em Python para gerenciar tarefas pelo terminal.

## Funcionalidades

- Adicionar tarefa
- Listar tarefas
- Marcar como concluída
- Remover tarefa

## Como usar

```bash
python main.py add "Estudar"
python main.py list
python main.py done 1
python main.py remove 1
